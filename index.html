About This Project

This site utilizes HTML 5.0 & an external CSS3 stylesheet for the template. Google Fonts have been referenced to give the content more character and a text shadow utilized to give more depth to the headers. Opacity was acheived using rgba with the background attribute on the layout sections.

There are 5 main semantic elements: Header, Aside, Nav, Section, and Footer.

  1. The header features the basic intro content for the page stating it's purpose.

  2. Aside holds a short blurb bio with an image that's been floated left with alt text.

  3. Nav contains all the the navigation for the site with a Unordered List that's had a rollover applied to it.

  4. Section contains all the content for the page with the index.html page containing a breakdown for the project.

  5. The footer holds copyright info.
